TanzaniaFamousAPI

An API providing information about famous Tanzanian personalities.

Usage

Make GET requests to the API endpoints to retrieve information about famous Tanzanians.

API Endpoints

  • GET /api/famous - Get all famous personalities
  • GET /api/famous/:id - Get a specific famous personality
  • GET /api/v1/query - Query famous personalities

Examples

Get Famous Person by ID

Example URL: GET https://tanzania-famous-api.netlify.app/api/famous/1 

{
  "success": true,
  "data": {
    "id": 1,
    "full_name": "Julius Kambarage Nyerere",
    "nickname": "Mwalimu",
    "date_of_birth": "1922-04-13",
    "date_of_death": "1999-10-14",
    "image_url": "https://aaregistry.org/wp-content/uploads/2009/09/julius-nyerere-225x300.jpg",
    "famous_for": "First President of Tanzania and a key figure in the country's independence."
  }
}

Get All Famous Personalities

Example URL: GET https://tanzania-famous-api.netlify.app/api/famous

Query Famous Personalities

Example URLs:

  • GET https://tanzania-famous-api.netlify.app/api/v1/query 
    {

  "limit": 10
