TanzaniaFamousAPI

An API providing information about famous Tanzanian personalities.

Usage

Make GET requests to the API endpoints to retrieve information about famous Tanzanians.

API Endpoints

  • GET /api/famous - Get all famous personalities
  • GET /api/famous/:id - Get a specific famous personality
  • GET /api/v1/query - Query famous personalities

Examples

Get Famous Person by ID

Example URL: GET https://api.example.com/api/famous/1 

{
  "success": true,
  "data": {
    "id": 1,
    "name": "Julius Nyerere",
    "dob": "1922-04-13",
    "description": "First President of Tanzania and African socialist leader."
  }
}

Get All Famous Personalities

Example URL: GET https://api.example.com/api/famous

Query Famous Personalities

Example URLs:

  • GET https://api.example.com/api/v1/query 
    {
  "success": true,
  "data": [/* All famous personalities */],
  "total": 10,
  "page": 1,
  "limit": 10
}
  • GET https://api.example.com/api/v1/query?search=John 
    {
  "success": true,
  "data": [/* Famous personalities with "John" in their name */],
  "total": 2,
  "page": 1,
  "limit": 10
}
  • GET https://api.example.com/api/v1/query?dob=1990-01-01 
    {
  "success": true,
  "data": [/* Famous personalities born on 1990-01-01 */],
  "total": 1,
  "page": 1,
  "limit": 10
}
  • GET https://api.example.com/api/v1/query?limit=10 
    {

  "data": [/* First 10 famous personalities */],
  "total": 10,

  "total": 10,

  "limit": 10
